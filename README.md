# wikipediaApp
A simple app that searches and lists wikipedia


### How to run this app

Ensure that you have your android/ios environment set up to run react-native apps

Run **git clone https://github.com/jeffngugi/wikipediaApp.git** to get the application into your machine.

cd ./wikipediaApp and run **yarn** to install packages

Run **yarn android** to start app in android or **yarn ios** to run the app in ios
